# Critic Handbook — 5-Dimension Creative Review (Rigid Checklist)

## When to Use
- Final quality gate in creative-agency pipeline (Stage 4)
- Any creative work needs qualitative review
- User asks "is this good?" about creative output

## Scoring Formula
- **PASS**: 80%+ total checks passed AND no "REJECT IF" criteria triggered.
- **WARN**: 60-79% total checks passed.
- **FAIL**: < 60% total checks passed OR any "REJECT IF" criteria triggered.

---

## 1. Craft (Technical Execution)
- [ ] Are there no technical artifacts or rendering errors?
- [ ] Is the resolution appropriate for the intended medium?
- [ ] Is all composition balanced and aligned?
- [ ] Is typography legible and properly tracked?
- [ ] Are all colors rendered according to design specifications?
- [ ] Is the file size optimized for the delivery platform?

**REJECT IF:**
- Contains visible pixelation, artifacts, or rendering errors.
- Fails accessibility standards (e.g., contrast).
- Incorrect file format or resolution for the specified deliverable.

## 2. Coherence (Internal Logic)
- [ ] Does the visual hierarchy guide the viewer’s eye appropriately?
- [ ] Does the copy directly support the visuals?
- [ ] Is the primary call to action (CTA) unambiguous?
- [ ] Is the tone of voice consistent throughout?
- [ ] Is the messaging focused on a single, clear objective?
- [ ] Are there any unnecessary elements that detract from the core message?

**REJECT IF:**
- Messaging contradicts the stated objective.
- Visuals and copy send conflicting signals.
- Primary CTA is missing or impossible to find within 3 seconds.

## 3. Brand Alignment (Identity)
- [ ] Is the color palette strictly adhering to the brand guidelines?
- [ ] Does the typography match the brand's approved font families?
- [ ] Is the voice and tone consistent with brand identity?
- [ ] Are brand assets (logo, icons) used correctly and not distorted?
- [ ] Does the style align with the brand’s positioning statement?
- [ ] Are any non-brand elements used in a way that violates guidelines?

**REJECT IF:**
- Logo is distorted, illegible, or misused.
- Uses unauthorized brand assets or color palettes.
- Violates core brand voice principles (e.g., too formal when it should be informal).

## 4. Emotional Resonance (Impact)
- [ ] Does the work elicit the intended emotional response from the target audience?
- [ ] Is the content relatable to the defined user persona?
- [ ] Does the work create a lasting impression or "stickiness"?
- [ ] Is the emotional approach appropriate for the stage of the customer journey?
- [ ] Does the work avoid being generic or purely informational?
- [ ] Are the emotional triggers clearly defined and executed?

**REJECT IF:**
- Content feels entirely cold, robotic, or devoid of personality.
- Uses emotionally inappropriate themes for the audience segment.
- Fails to differentiate from direct competitors emotionally.

## 5. Creative Ambition (Originality)
- [ ] Does the work demonstrate a fresh or original approach to the brief?
- [ ] Does it push beyond basic template-driven layouts?
- [ ] Is the risk level appropriate for the brand's positioning?
- [ ] Does it avoid tired clichés or overused tropes?
- [ ] Is there a clear, unique hook or angle?
- [ ] Does it challenge the status quo within the parameters allowed?

**REJECT IF:**
- The work is a direct copy of a competitor's asset.
- Relies heavily on overused stock imagery or generic templates.
- Lacks a distinct point of view or creative spark.
