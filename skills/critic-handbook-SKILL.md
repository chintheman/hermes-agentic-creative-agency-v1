# Critic Handbook — 5-Dimension Creative Review Methodology

## When to Use

- Final quality gate in creative-agency pipeline (Stage 4)
- Any creative work needs qualitative review
- User asks "is this good?" about creative output
- Training / calibrating creative quality standards

## The 5 Dimensions

### 1. Craft (Technical Execution)
How well-made is the work? Is the technical execution polished?

| Score | Description |
|---|---|
| 10 | Exhibition-grade — flawless execution |
| 8-9 | Professional — minor refinements possible |
| 6-7 | Competent — technically sound but unrefined |
| 4-5 | Below professional standard — noticeable flaws |
| 1-3 | Amateur — fundamental execution issues |

**Checklist:**
- [ ] No obvious technical flaws (artifacts, broken layout, typos)
- [ ] Resolution/quality appropriate for medium
- [ ] Attention to detail in finishing
- [ ] Tools used effectively

### 2. Coherence (Internal Logic)
Does the work make sense as a whole? Does the message, visual, and structure work together?

| Score | Description |
|---|---|
| 10 | Perfect narrative — every element serves the message |
| 8-9 | Clear and unified — minor disconnects |
| 6-7 | Mostly coherent — some elements feel disconnected |
| 4-5 | Fragmented — message and medium don't align |
| 1-3 | Incoherent — no clear through-line |

**Checklist:**
- [ ] Copy and visuals tell the same story
- [ ] Message hierarchy is clear (what's most important?)
- [ ] Format serves the objective
- [ ] No conflicting signals

### 3. Brand Alignment (Identity Consistency)
Does this feel like it belongs to the brand? Does it use brand assets correctly?

| Score | Description |
|---|---|
| 10 | Undeniably the brand — couldn't be anyone else |
| 8-9 | Strong brand presence — minor palette/tone deviations |
| 6-7 | Recognizable — but doesn't fully leverage brand equity |
| 4-5 | Weak brand connection — could be any brand |
| 1-3 | Brand breach — contradicts brand identity |

**Checklist:**
- [ ] Palette matches brand DNA (90%+ = PASS)
- [ ] Tone matches brand voice adjectives
- [ ] Typography follows brand specs
- [ ] Positioning aligned with brand differentiators

### 4. Emotional Resonance (Audience Impact)
Does the work connect emotionally? Does it make the audience feel something?

| Score | Description |
|---|---|
| 10 | Powerful — memorable, shareable, evocative |
| 8-9 | Engaging — connects with target audience |
| 6-7 | Pleasant — competent but doesn't move |
| 4-5 | Flat — misses the emotional mark |
| 1-3 | Repellant — creates wrong emotional response |

**Checklist:**
- [ ] Appropriate emotional register for audience
- [ ] Creates a reaction (not just information delivery)
- [ ] Memorable quality
- [ ] Avoids cringe or manipulation

### 5. Creative Ambition (Originality & Risk)
Does the work take creative risks? Is it doing something interesting?

| Score | Description |
|---|---|
| 10 | Breakthrough — genuinely novel approach |
| 8-9 | Distinctive — stands out in its category |
| 6-7 | Solid — good execution of established patterns |
| 4-5 | Safe — follows formula without adding |
| 1-3 | Derivative — copies existing work too closely |

**Checklist:**
- [ ] Fresh approach (not just template-filling)
- [ ] Appropriate risk level for brief and brand
- [ ] Adds something to the conversation
- [ ] Avoids cliché

## The Gate Decision Matrix

| Avg Score | Lowest Dimension | Verdict |
|---|---|---|
| ≥ 7.0 | ≥ 6.0 | SHIP ✅ |
| ≥ 7.0 | 4.0-5.9 | FIX 🔧 |
| 5.0-6.9 | — | REVISE 🔄 |
| < 5.0 | — | BLOCK 🛑 |
| — | < 4.0 | BLOCK 🛑 |

## Review Output Template

```markdown
### Critic Handbook Review

**Craft:** 8/10 — Clean execution, minor shadow refinement needed
**Coherence:** 9/10 — Strong narrative, copy and visual align
**Brand Alignment:** 9/10 — 94% palette match, tone is on-brand
**Emotional Resonance:** 7/10 — Warm and approachable, could be more memorable
**Creative Ambition:** 6/10 — Solid but safe execution of established format

**Average: 7.8/10 — SHIP ✅**

**Key Strengths:**
- Excellent brand integration
- Clear narrative flow

**Growth Areas:**
- Creative ambition could push further
- Emotional hook could be stronger
```
